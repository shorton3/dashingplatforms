Detailed Description

The purpose of this class is to create an ACE_Service_Object subclass to allow dynamic loading/unloading of our application processes. This class acts as a bridge pattern between the Service Configurator(Service Object) and the ACE Process_Manager

Each instance of this class corresponds to a configured Service/Process in the system.
