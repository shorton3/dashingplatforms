Detailed Description

LocalSMBuffer class is a wrapper for the MessageBuffer's raw unsigned char buffer so that it can be passed through the shared memory queue with a position independent pointer.

LocalSMBuffer assumes that all shared memory passed message will be of length MAX_MESSAGE_LENGTH.
