const string & SystemInfo::getLocalNEID ( )
static

Return the local card's NEID (Network Element Id). This method reads the Local NEID from the environment the first time it is called. After that, it caches the value and just returns the cached value (so a complete restart is needed if this value needs to change).

Returns
the 9 digit NEID string, or it returns "000000000" upon failure if it could not be taken from the environment, its length is not 9, or its characters are not digits.
