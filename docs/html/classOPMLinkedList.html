Detailed Description

This utility class implements a doubly linked list with both a head and tail pointers for storing OPMBase objects. All list operations are done by manipulating pointers since OPMBase stores the references for the 'NEXT' and 'PREV' pointers.

This class is currently NOT THREAD SAFE.
