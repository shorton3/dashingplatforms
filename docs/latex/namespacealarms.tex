\subsection{Detailed Description}
Interface for sending alarm notification from the E\+MS to its clients. 

Also allows clients to interrogate the E\+MS for alarm inventory, as well as resend/resync outstanding alarms, and apply filters. 

Beware of using a symbol named \textquotesingle{}alarm\textquotesingle{} as this conflicts with symbols in $<$unistd.\+h$>$
